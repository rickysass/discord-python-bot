You'll need to install a few things in this order most likely:

- Install the newest version of python
- Verify pip is installed on your machine
- Run 'pip install -U discord.py python-dotenv emoji'
- Create a .env file with a variable inside of it: 'DISCORD_BOT_TOKEN={Ask the owner for the token}'

In command line, run the bot with 'python3 bot.py'
To close the bot, either type !shutdown in the discord or CTRL+C in the command line.
